# 🍲 chihqiang/soft - 家常菜式 Homebrew 仓库

常用软件，一键安装！
没啥复杂，装完即用，码农效率蹭蹭飙升，老板都夸你“真快”！

## 添加仓库

```bash
brew tap chihqiang/soft
```

## 查看本仓库的详细信息

```bash
brew tap-info chihqiang/soft
```

## 移除本仓库

```bash
brew untap chihqiang/soft
```

## 清理缓存

```bash
brew cleanup chihqiang/soft
```

## 常见软件

### [sshpass](https://sourceforge.net/projects/sshpass/)

```bash
brew install chihqiang/soft/sshpass
```

### [sshfwd](https://github.com/chihqiang/sshfwd)

```bash
brew install chihqiang/soft/sshfwd
```
